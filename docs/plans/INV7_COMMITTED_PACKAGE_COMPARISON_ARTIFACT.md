## Gap

**Current:** `make compare-inv7-packages` and
`qc_cli.py compare-inv7-packages` can compare schema_version=1 INV-7 result
packages, but the repo has no committed comparison artifact that demonstrates
the output over existing benchmark packages.

**Target:** Commit a small comparison artifact under `docs/benchmarks/` that
compares the current built-in v2 canary, the external held-out smoke artifact,
and the 10-fixture held-out live v1 artifact. Include the generated JSON report
and a README with exact commands, input rationale, result summary, and
claim-discipline caveats.

**Why:** This gives Brian and future agents a durable output surface to inspect
without rerunning commands, and it proves the comparison workflow works on real
committed artifacts rather than only synthetic tests. It remains a
measurement/provenance artifact, not robustness evidence.
